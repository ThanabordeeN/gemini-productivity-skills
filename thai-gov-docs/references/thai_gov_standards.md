# ระเบียบสำนักนายกรัฐมนตรีว่าด้วยงานสารบรรณ (ฉบับย่อสำหรับบันทึกข้อความ)

## 1. การตั้งค่าหน้ากระดาษ (Page Layout)
- **ขอบบน (Top):** 2.5 เซนติเมตร
- **ขอบล่าง (Bottom):** 2.0 เซนติเมตร
- **ขอบซ้าย (Left):** 3.0 เซนติเมตร
- **ขอบขวา (Right):** 2.0 เซนติเมตร

## 2. ตัวอักษร (Fonts)
- **ชื่อฟอนต์:** TH Sarabun New
- **ขนาดปกติ:** 16 พอยท์ (docxjs: size 32)
- **หัวข้อ "บันทึกข้อความ":** ตัวหนา ขนาด 24 พอยท์ (docxjs: size 48) วางกึ่งกลาง
- **หัวข้อ "ส่วนราชการ", "ที่", "วันที่", "เรื่อง":** ตัวหนา ขนาด 16 พอยท์

## 3. ตราครุฑ
- **ขนาด:** สูง 1.5 เซนติเมตร (แบบเล็กสำหรับบันทึกข้อความ)
- **ตำแหน่ง:** วางชิดขอบซ้ายในบรรทัดเดียวกับคำว่า "บันทึกข้อความ"

## 4. ระยะห่างและการย่อหน้า
- **ย่อหน้า:** 2.5 เซนติเมตร
- **ระยะบรรทัด:** Single Space (ปกติ)
- **เส้นคั่น:** มีเส้นทึบคั่นระหว่างส่วนหัวและเนื้อหา
